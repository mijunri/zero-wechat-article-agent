# 头条号民生热点内容生产流水线 v2.0

本 skill 定义头条号民生热点内容生产的完整工作流，每日自动产出 3 篇民生/社会类文章。

## 为什么选民生类？

头条用户群体：35-55岁，三四线城市为主，更关心：
- 社会热点和真实案例
- 家庭、情感、婚姻话题
- 健康、教育、职场问题
- 奇葩新闻和纠纷事件

民生类内容的优势：
- 话题接地气，用户有代入感
- 容易引发讨论和转发
- 标题自带戏剧性（"18年槟榔""黄金当废品"）

## 第一步：下载所有平台热搜数据

使用 `hot-topics` skill 获取所有平台的热搜榜，保存为 Markdown 格式。

**执行方式**：
```
用 hot-topics 获取所有平台热搜，转换成 Markdown，保存到 work/YYYY-MM-DD/1_hot/ 目录
```

**要求**：
- 平台：全部（微博、知乎、百度、抖音、今日头条、B站）
- 数量：每个平台至少 20 条
- 输出目录：`work/YYYY-MM-DD/1_hot/`
- 文件命名：`{platform}_hot.md`
- 格式：Markdown 表格（排名 | 标题 | 热度）

**验证标准**：
- ✅ 所有 6 个平台数据都已下载
- ✅ 文件保存在 `work/YYYY-MM-DD/1_hot/` 目录
- ✅ 每个文件是 Markdown 格式，包含排名、标题、热度
- ✅ 数据时间戳为今日

**输出格式示例**：

## 第二步：筛选"最愿意点开"的话题

从已下载的 `work/YYYY-MM-DD/1_hot/toutiao_hot.md` 中筛选**低文化中年人最愿意点开**的话题。

**核心目标**：点击率（CTR）优先，不是"民生类"，而是"想点开看的"。

**执行方式**：
```
从 work/YYYY-MM-DD/1_hot/toutiao_hot.md 读取头条热搜
筛选"最愿意点开"的话题，按热度排序取前 5 个
每个话题保存一个 md 文件到 work/YYYY-MM-DD/2_filter/
```

**筛选标准（点击驱动）**：

1. **好奇心驱动** - "到底咋回事？""真的假的？"
   - 反常识：树冒烟、冰箱炸死人、鸳鸯锅学游泳
   - 悬念：不看不知道真相

2. **利己相关** - "这事儿跟我有关""我也能遇到"
   - 钱：公积金、社保、退休金、股市
   - 健康：中毒、疾病、食品安全
   - 家庭：婚姻、彩礼、孩子、老人

3. **吃瓜心理** - "别人倒霉/干蠢事，我看看热闹"
   - 奇葩操作：女子拿着鸳鸯锅学游泳
   - 别人倒霉：失独妈妈、被网暴

4. **数字冲击** - 具体数字更有吸引力
   - 30年社保、240万索赔、20岁厨师、两天两夜

5. **负面但与自己无关** - 可以安全吃瓜
   - 诈骗、事故、奇葩新闻（不是自己身上）

**排除（低点击率）**：
- 宏观政策："三北"治理、AI大牛市（离自己太远）
- 纯政治：黄岩岛、台湾、外交（不关心）
- 纯娱乐：明星官宣、歌手节目（竞争太激烈）
- 纯体育：国足、NBA（非球迷不点）

**输出格式**：每个话题一个 md 文件

```markdown
# 远看树冒烟近看全是虫

## 基本信息
- 标题：远看树冒烟近看全是虫
- 热度：12,001,716
- 排名：第4名
- 来源：今日头条热搜

## 为什么中年人愿意点开？

### 好奇心驱动
- "树怎么会冒烟？" → 反常识，想知道真相
- "近看全是虫" → 有画面感，有点恶心但想看

### 反差感
- 远看：树冒烟（像是着火了）
- 近看：全是虫子（完全不是那么回事）

### 传播性
- 话题短，好记，容易跟人说
- "你看到那个树冒烟的新闻没？"

## 核心要素
- 视觉冲击：树冒烟 → 全是虫
- 反转：以为是火灾，结果是虫子
- 话题性：可以跟人聊天当谈资

## 写作角度
- 开头：树怎么会冒烟？
- 展开：走近一看，头皮发麻
- 科普：这是什么虫？为什么会这样？
- 收尾：大自然的神奇（或诡异）
```

## 注意事项

1. **搜索先行**：写稿前必须10轮搜索，不能仅凭热搜标题写稿
2. **点击优先**：不是"民生类"，而是"最愿意点开" — 好奇/利己/吃瓜/反常识
3. **质量优先**：宁缺毋滥，搜索结果不足时跳过该话题
4. **七步迭代**：每篇文章必须经过思考→大纲→初稿→审稿→修改→再审→最终
5. **活人感**：避免 AI 味，用口语化表达
6. **吸睛标题**：标题要有悬念/情绪/代入感，利用事件本身的戏剧性
