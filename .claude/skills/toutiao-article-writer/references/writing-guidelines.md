# 头条号写作规范

## 目标读者

- **年龄**：35-55 岁
- **地域**：三四线城市为主
- **特点**：低文化、手机阅读、时间碎片化
- **偏好**：八卦、奇闻、与己相关、反常识

---

## 标题技巧

### 好标题的 5 个特征

1. **有数字**：30年、20岁、240万、两天两夜
2. **有反转**：医生中毒进自己科室、树冒烟全是虫
3. **有悬念**：到底咋回事？真的假的？
4. **有情绪**：炸死、中毒、昏迷、全家
5. **有代入**：这事儿你也可能遇到

### 标题公式

```
{人物/主体} + {意外事件} + {数字/细节}
例：云南医生吃菌子中毒住进自己医院：全家连续三年中招
```

```
{反常识现象} + {真相}
例：远看树冒烟近看全是虫，走近看头皮发麻
```

```
{利己话题} + {反差}
例：同样交社保30年，为何退休金差这么多？
```

### 标题避坑

- ❌ 太长：超过 30 字
- ❌ 太虚："震惊！""必看！"
- ❌ 太官方："关于...的通知"
- ❌ 预告："明天告诉你"（骗点击）

## 开头技巧

### 1. 直接甩场景

```
张万福醒来的时候，发现自己躺在自己医院的病床上。
护士问他："医生，你没事吧？"
他这才反应过来——自己昏迷了整整48小时。
```

### 2. 用数字说话

```
20岁的印度厨师正在厨房干活，冰箱突然"砰"的一声炸了。
碎片插满全身，送医院抢救无效死亡。
```

### 3. 制造悬念

```
远看树在冒烟，像着火了一样。
走近一看，头皮发麻——全是虫。
```

## 正文写作

### 短句原则

- 每句 10-20 字
- 一段不超过 3 行
- 手机阅读友好

### 信息密度

- ❌ 注水："在这篇文章中，我们将探讨..."
- ❌ 过渡："首先""其次""最后"
- ❌ 总结："综上所述""总而言之"
- ✅ 直接摆事实
- ✅ 用对话
- ✅ 用场景

### 活人感

```
❌ AI味：
"这说明该事件对社会造成了深远影响。"

✅ 活人感：
"这事一出，村里人都议论开了。"
```

### 段落节奏

- 长短结合，不要都一样
- 单句成段，制造强调
- 用对话打破大段文字

---

## 情绪节奏

### 典型结构

```
开头：炸（惊讶/好奇）
  ↓
中间：魔幻/反转（故事展开）
  ↓
结尾：观点/警示（情绪落点）
```

### 结尾技巧

1. **警示型**：别为了追求口感把命搭进去
2. **观点型**：命比口感重要，真的不值得
3. **反问型**：这事你怎么看？
4. **共情型**：换作是你，你会怎么做？

## 去AI味检查清单

写完文章后逐项检查：

- [ ] 没有"这说明了""值得注意的是"
- [ ] 没有"首先/其次/最后"
- [ ] 没有"综上所述""总而言之"
- [ ] 段落长度有变化
- [ ] 有对话、场景、数字
- [ ] 结尾有观点，不是道德说教
- [ ] 标题有数字/反转/悬念
- [ ] 开头直接，没有废话

## 信息框使用

### info-box（蓝色/灰色）

呈现关键信息、事实、数据。

```
关键信息：
• 事件时间：2026年6月5日
• 当事人：张万福，主任医师
• 症状：昏迷2天、连续失忆
```

### highlight-box（黄色）

强调重点内容、金句。

```
医生的原话：
"菌子能不吃的就不要吃，不会做的不要做。"
```

### warning-box（红色）

提醒、警示、安全提示。

```
提醒：
吃菌子出现幻觉、恶心、呕吐，立即催吐就医。
别硬扛，命要紧。
```

## 字数要求

- **推荐**：900-1200 字
- **最低**：不低于 600 字
- **上限**：不超过 1500 字

---

## 小标题使用

- 2-4 个小标题为宜
- 小标题要有悬念/数字
- 小标题之间要有递进

```
❌ 一、事件经过
   二、专家分析
   三、温馨提示

✅ 医生醒来发现自己躺在医院
   连续三年全家中毒，魔幻不？
   菌子不能凭感觉，命要紧
```
